Listas y Pilas 📚

## 🛠️ Contenido del Proyecto
El proyecto incluye la implementación funcional de:
* **Pilas (Stacks):** Uso del concepto LIFO para el manejo de datos. * **Listas Simples:** Estructuras de nodos enlazados para almacenamiento dinámico. * **Listas Doblemente Enlazadas:** Implementación de nodos con punteros anteriores y siguientes. 
## 🚀 Tecnologías Utilizadas
* **Lenguaje:** Java ☕
* **IDE:** Apache NetBeans
